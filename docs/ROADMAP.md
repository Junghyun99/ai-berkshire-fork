# ai-berkshire Roadmap

## P0: 단기 (1-2개월)

### 한국/중국 데이터 소스 연동
- DART OpenAPI, akshare, East Money 등 무료 데이터 소스 연동
- 한국 주식·중국 A주의 재무 데이터, 시세 커버
- 기존 Skill은 수정 불필요, 데이터 계층 확장만으로 가능

## P1: 중기 (3-6개월)

### HTML 보고서 출력
- Markdown 기반 위에 HTML 보고서 형식 추가
- 다크 모드, 내비게이션 바, 차트 시각화 지원
- 보고서의 전파성과 읽기 경험 향상

### 다단계 깊이 모드
- `lite`: 5분 속성 판단, 밸류에이션 구간과 핵심 결론을 빠르게 제시
- `standard`: 현재 기본 모드, 완전한 멀티 Agent 리서치
- `deep`: 교차 검증과 역사적 유추를 추가한 기관급 깊이

### 다종목 횡단 비교
- 2-4개 종목의 동일 차원 횡단 대결 지원
- 동종 산업 기업 밸류에이션 벤치마킹
- 비교 매트릭스와 우선순위 제안 출력

## P2: 장기 (6개월+)

### 테스트 커버리지
- 핵심 도구(financial_rigor.py 등)에 단위 테스트 추가
- Skill 출력에 회귀 테스트 추가
- 반복 개선이 기존 기능을 깨뜨리지 않도록 보장

### 포트폴리오 수준 분석
- 보유 포트폴리오 건강도 평가
- 산업/지역 집중도 분석
- 상관관계 리스크 탐지
