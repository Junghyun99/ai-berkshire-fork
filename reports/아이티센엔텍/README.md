## 핵심 결론

- **회사**: 구 쌍용정보통신(국내 SI 1호, 1981). 공공·국방·금융 SI. 2020년 아이티센그룹 편입
- **정량**: 정상화 PER ~3.3배, PBR 0.61배, 순현금 152~530억(시총 606억) — 명백한 저평가
- **정성**: 무배당 3년, 다층 지배구조(강진모→글로벌→씨티에스→엔텍 38%), 일 거래대금 ~3천만원 — 가치 실현 경로 부재
- **판단**: 승부처는 사업이 아니라 **지배구조 촉매**. 배당·자사주·공정 재편 신호 확인 전까지 관망
