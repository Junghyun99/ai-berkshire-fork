# 경영진 심층 연구: 주식을 사는 것은 사람을 사는 것이다

$ARGUMENTS 에 대해 경영진 심층 연구를 수행한다.

**지원하는 입력 형식**: `회사명` 또는 `인물명 회사명`. 예: `메이투안`, `왕싱 메이투안`, `젠슨 황 엔비디아`

> "주식을 사는 것은 사람을 사는 것이다. 신뢰할 수 있는 사람을 찾아 장기 보유하라." — 돤융핑
>
> "경영진을 평가하려면 아무도 보지 않을 때 그들이 무엇을 하는지 봐야 한다." — 버핏

## 설계 이념

대부분의 투자 분석에서 경영진 평가는 표면에 머문다: 이력, 지분율, 보수. 그러나 버핏은 **경영진과 식사하며 대화하는 데** 많은 시간을 쓰고, 리루는 **자기 투자의 본질은 사람에 대한 투자**라고 말하며, 돤융핑은 **주식을 사는 것은 사람을 사는 것**이라고 말한다.

이 Skill은 `/investment-research` 5단계 경영진 평가의 **심화판**이다. 표준 투자 리서치에서 경영진 점수가 불확실하거나(★★★ 이하), 경영진이 핵심 투자 논리일 때 이 Skill로 심층 연구한다.

AI는 경영진과 식사할 수 없지만, 공개 정보로 다음을 할 수 있다:
- **경영진의 말과 행동이 일치하는지 추적** (약속 vs 이행)
- **모든 중대 자본 배분 결정의 수익률 분석**
- **어려운 시기의 결정에서 품격을 추론**
- **직원/파트너/고객의 피드백으로 측면 검증**

핵심 인물 확인 후, Task 도구로 여러 백그라운드 Agent를 **병렬** 실행해 아래 데이터를 수집한다:
1. Agent 1: CEO의 공개 발언과 예측 기록 (주주 서한, 어닝콜, 인터뷰, 소셜미디어)
2. Agent 2: 자본 배분 결정 기록 (M&A, 자사주 매입, 배당, 신사업 투자)
3. Agent 3: 지배구조와 보수 (지분 구조, 특수관계자 거래, 임원 보수)
4. Agent 4: 측면 검증 정보 (직원 평가, 고객 피드백, 업계 평판)

- 산업 트렌드에 대한 CEO의 이해는 시장 추종인가 독립적 사고인가?

| 조직 역량 | 인재를 끌어들이고 붙잡을 수 있는가? | |

### 3단계: 정직성 평가 (가장 중요)

**버핏**: "우리는 세 가지 자질을 찾는다: 정직, 지성, 에너지. 첫 번째가 없으면 나머지 둘이 당신을 죽인다."

#### 3.1 약속 vs 이행 추적

지난 3년의 어닝콜, 주주 서한, 공개 인터뷰에서 경영진이 한 **구체적 약속**을 추출한다:

| # | 시기 | 약속 내용 | 약속한 자리 | 이행 상황 | 평가 |
|---|------|---------|---------|---------|------|
| 1 | | "2025년에 X 사업 흑자를 달성하겠다" | 2024 연간 어닝콜 | | ✅/⚠️/❌ |
| 2 | | "X억 달러 자사주 매입 계획" | 2024 주주 서한 | | ✅/⚠️/❌ |

**이행률 통계**:

| 약속 이행률 | 평가 |
|:---------:|------|
| >80% | 우수 — 말한 것은 해낸다 |
| 60-80% | 합격 — 큰 방향은 맞으나 실행에 편차 |
| 40-60% | 우려 — 약속은 많고 이행은 부족 |
| <40% | 심각한 문제 — 신뢰 불가 |

#### 3.2 어려운 시기의 모습

회사가 역사상 겪은 중대 위기/어려움(주가 폭락, 실적 miss, 규제 충격, 경쟁 심화)을 검색해 경영진의 대응을 분석한다:

| 위기 이벤트 | 시기 | 경영진의 반응 | 사후 평가 |
|---------|------|-----------|-------------|

**관찰 포인트**:
- 능동적으로 소통했는가, 회피했는가?
- 내부에서 원인을 찾았는가, 외부에 책임을 돌렸는가?
- 그 기회에 어렵지만 옳은 일을 했는가, 단기적으로 시장 비위를 맞췄는가?

#### 3.3 이해관계자에 대한 태도

| 이해관계자 | 경영진의 태도 | 증거 | 평가 |
|-----------|-----------|------|------|
| 주주 | 존중/무시/이용 | | |
| 직원 | 선대/착취/방치 | | |
| 고객/사용자 | 고객 중심/단기 착취 | | |
| 파트너/공급사 | 공정한 협력/극단적 단가 후려치기 | | |
| 규제기관/사회 | 준법 협조/편법 줄타기 | | |

**리루**: "이해관계자에 대한 태도가 기업의 장기 생명력을 결정한다. 단기 착취는 효율을 올리지만 장기적으로 생태계를 훼손한다."

#### 돤융핑의 "사람을 산다" 기준

> 아래 세 질문에 답하라:
> 1. **이 사람은 정직한가?** (솔직하고, 주주의 것을 탐하지 않는가)

> 3. **이 사람에게 내 돈을 10년 맡길 의향이 있는가?**

> 셋 다 "예" = ★★★★★ (5점)
> 앞의 둘이 "예" = ★★★★ (4점)
> 첫 번째만 "예" = ★★★ (3점)
> 첫 번째가 "예"가 아님 = ★ (1점, 투자하지 않음)

## 핵심 원칙

- **정직성은 단독 부결 사항이다** — 능력 부족은 배울 수 있지만, 품격의 문제는 고칠 수 없다
- **말이 아니라 행동을 본다** — 경영진이 무엇을 말했는지는 중요하지 않다. 무엇을 했는지가 중요하다
- **어려움 속에서 진실을 본다** — 순풍일 때는 누구나 좋은 CEO다. 역풍일 때 진짜 실력이 드러난다
- **자본 배분은 최종 시험이다** — 돈을 버는 것은 쉽다. 번 돈을 잘 배분하는 것이 어렵다
- **경영진과 사랑에 빠지지 마라** — 객관성을 유지하라. 당신이 존경하는 사람도 큰 실수를 할 수 있다
