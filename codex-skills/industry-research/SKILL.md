# 산업 투자 리서치: 밸류체인 전경 스캔 + 4대 거장 개별 종목 분석 프레임워크

## 리서치 목표

하나의 투자 테마/논리 사슬에서 출발해 다음을 완성한다:
1. 투자 논리 사슬의 모든 고리 검증
2. 완전한 밸류체인 전경도 작성
3. 전 세계 모든 상장사 스캔 (한국/중국 A주/홍콩/미국/기타)

## 1단계: 투자 논리 사슬 구축과 검증

### 1.1 논리 사슬 그리기

```

    → 수요 B 유발

```

### 1.2 고리별 검증
논리 사슬의 각 화살표에 의문을 제기하고 증거를 찾는다:

| 고리 | 핵심 가정 | 검증 방법 | 데이터 출처 |
|------|---------|---------|---------|
| A→B | | 산업 데이터/전망 검색 | |
| B→C | | 수급 분석 검색 | |
| C→D | | 실제 사례/계약 검색 | |

### 1.3 "이미 발생한 검증 이벤트" 찾기
논리 사슬을 뒷받침하는 **이미 계약/실행된 실제 상업 이벤트** (예측이 아니라)를 나열한다. 예: 대기업의 구매 계약, 정책 문서, 산업 보고서 등.

## 2단계: 밸류체인 전경도 작성

### 2.1 밸류체인 구조 그리기
산업을 상류→중류→하류→보조 단계로 분해한다. 예:
```
상류: 원자재/자원 채굴 → 소재 가공/정제
중류: 핵심 장비 제조 → 시스템 통합/엔지니어링 → 신기술 R&D
하류: 운영/서비스 → 최종 고객
보조: 검사/인증 → 유지보수 서비스 → 금융 상품 (ETF/신탁)
```

### 2.2 단계별 "사업 특성" 식별
각 단계에 표기한다:

| 단계 | 비즈니스 모델 | 매출총이익률 구간 | 경쟁 구도 | 장벽 유형 | 경기민감도 |
|------|---------|-----------|---------|---------|--------|
| | 자원 판매/장비 판매/서비스 판매/임대 수익 | | 독점/과점/완전 경쟁 | 자원/라이선스/기술/규모 | 강/중/약 |

### 2.3 "병목 단계" 표시
밸류체인에서 공급이 가장 빡빡하고, 대체가 가장 어려우며, 이익률이 가장 높은 단계를 식별한다 — 최고의 투자 대상은 대개 여기에 있다.

## AI 리서치 편향 자각: 산업 리서치의 특수한 함정

산업 리서치에서 AI 데이터 편향은 독특한 방식으로 증폭된다:

**산업 수준의 편향**:
| 편향 유형 | 양상 | 대응 |
|---------|------|------|
| 성숙 산업 선호 | 전통 산업(은행/에너지/소비재)은 자료가 매우 많아 AI 분석이 "더 확실해" 보임 | 확실성은 비즈니스 모델에서 나오지, 리포트 수에서 나오지 않는다 |
| 신흥 산업 저평가 | 새 산업(AI 응용/합성생물학 등)은 자료가 적어 AI 분석이 보수화됨 | "현재 데이터"가 아니라 "최종 국면 사고"로 산업 가치를 판단 |
| 선두 기업 선호 | 대기업 자료가 소기업보다 훨씬 많아 AI가 선두 기업을 추천하는 경향 | 소기업이 더 좋은 위험 대비 수익을 가질 수 있다. AI 분석 분량이 짧다고 무시하지 마라 |
| 상장사 선호 | 상장사만 스캔하면 밸류체인의 핵심 비상장 플레이어를 놓침 | 반드시 비상장사도 검색하고 "미래 IPO 후보"로 표기 |
| 영어 선호 | AI는 영어 자료 처리에 더 강해 아시아 시장 플레이어를 저평가할 수 있음 | 반드시 영어와 현지어 정보원을 함께 검색 |

**밸류체인 스캔의 반편향 조치**:
1. 각 단계에서 "AI가 쉽게 찾는 회사"뿐 아니라 "비주류지만 우량할 수 있는 종목"을 능동적으로 검색
2. 정보가 희소한 소형주에 대해 분석 분량이 짧다고 추천도를 낮추지 않는다 — 보고서 길이가 아니라 핵심 질문(사업 본질, 해자, 경영진)으로 평가
3. 최종 보고서에 각 회사의 "정보 충분도"(A/B/C등급)를 표기해 AI 분석의 신뢰 수준을 독자가 알게 한다

- **Tier 1**: 대형주, 순수 종목, 산업 선두
- **Tier 2**: 중형주, 순수 또는 고비중, 세부 분야 선두
- **Tier 3**: 소형주, 개발 단계, 고위험 고탄력
- **Tier 4**: 관련 사업을 보유한 다각화 대기업

## 4단계: 단계별 선두 기업 4대 거장 분석

각 밸류체인 단계의 **Tier 1과 Tier 2 기업**에 아래 분석을 실행한다 (Tier 3/4 기업은 간단한 코멘트면 충분):

### 4.1 사업 본질 (돤융핑)
- 이 회사가 밸류체인에서 무엇을 하는지 한 문장으로 정의
- 매출 구조와 성장률
- 매출총이익률/순이익률 수준과 추세
- 현금흐름 특성
- **되물음**: 이것은 좋은 사업인가? 왜?

### 4.2 해자 (버핏)
다섯 가지 해자로 채점 (★1-5):

| 해자 | 강도 | 증거 |
|--------|------|------|
| 브랜드/가격결정력 | | |
| 전환 비용 | | |
| 네트워크 효과 | | |
| 규모의 경제 | | |
| 기술/라이선스 장벽 | | |

**되물음**: 10년 후에도 해자가 남아 있을까?

### 4.3 리스크 (멍거)
- 이 회사는 어떻게 실패할 가능성이 가장 큰가?
- 최악의 시나리오에서 얼마짜리인가?
- 똑똑한 사람들은 왜 안 사는가?

### 4.4 경영진 (돤융핑+버핏)
- CEO/창업자는 누구인가? 핵심 의사결정 기록
- 지분율과 이해관계 정렬
- 간단 평가 (A/B/C등급)

### 4.5 밸류에이션 스냅샷
- 현재 PER/PSR/EV/EBITDA
- 같은 단계 경쟁사와 비교
- 간단 평가: 비쌈/적정/저렴

### 4.6 추천도
★1-5로 표기:
- ★★★★★ = 핵심 비중 후보
- ★★★★☆ = 위성 비중 후보
- ★★★☆☆ = 관찰 목록
- ★★☆☆☆ = 고위험 옵션
- ★☆☆☆☆ = 비추천
