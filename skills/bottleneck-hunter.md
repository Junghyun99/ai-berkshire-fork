# 공급망 병목 사냥꾼: AI 기반 글로벌 밸류체인 병목 차익거래

$ARGUMENTS 슈퍼 트렌드에 대해 공급망 병목 스캔과 차익거래 기회 발굴을 실행한다.

## 핵심 이념

"AI야, 어떤 주식을 추천해?"라고 묻지 않고, "이 트렌드가 계속 확장되면 어느 고리가 먼저 부족해지는가?"라고 묻는다.

전통적 투자 리서치는 선두 기업과 알려진 분야를 본다. 이 시스템은 거꾸로다: **물리적 공급망의 목구멍 위치에서 출발해, 아무도 주목하지 않지만 일단 품절되면 산업 전체가 멈춰서 기다려야 하는 회사를 찾는다.**

초과수익의 원천: 1층 병목(GPU, HBM, 전력)은 이미 충분히 가격에 반영됐다. 진짜 alpha는 **2층, 3층**에 있다 — 광모듈, 레이저, InP 기판, SOI 웨이퍼, 에피택시 장비, 웨이퍼 레벨 테스트, IC 기판, 특수 유리섬유 등.

## 1단계: 슈퍼 트렌드 확인

### 1.1 트렌드 선별 기준

작은 유행에서 환상을 찾지 않는다. 아래 조건을 전부 충족하는 슈퍼 트렌드만 쫓는다:

| 기준 | 요구사항 | 검증 방법 |
|------|------|---------|
| 지속성 | 최소 3-5년의 확실한 성장 | 산업 전망, 자본지출 계획 검색 |
| 물리성 | 실제 하드웨어/소재/장비 구축 필요 | "소프트웨어 업그레이드"와 "물리적 증설" 구분 |
| 규모성 | 글로벌 자본지출 >500억 달러/년 | 선두 플레이어의 capex 가이던스 검색 |
| 가속성 | 수요 증가 속도 > 공급 증설 속도 | 수요 증가율 vs 생산능력 확장 계획 비교 |

### 1.2 현재 추적 중인 슈퍼 트렌드 목록

실행할 때마다 갱신. 초기 목록:

1. **AI 인프라 구축** — 데이터센터, GPU 클러스터, 네트워크 인터커넥트, 전력
2. **에너지 전환** — 원전 재가동, 전력망 업그레이드, 에너지 저장
3. **국방 현대화** — 서방 국방비 상승 사이클, 공급망 재편
4. **반도체 재산업화** — 미국·유럽·일본의 보조금 공장 건설, 장비/소재 병목
5. **우주 경제** — 위성 인터넷, 발사 횟수 급증

사용자가 특정 트렌드(예: "AI 인프라")를 지정했으면 그 트렌드에만 집중한다.

### 1.3 트렌드 검증 출력

```

핵심 동력: (한 문장)
이미 발생한 검증 이벤트 (최소 3개):
  1. [날짜] [이벤트] [출처]
  2.
  3.
자본지출 규모: 글로벌 약 $XX억/년, 성장률 YY%
수급 격차 판단: 수요 증가 속도 > 공급 증설 속도? 예/아니오/불확실
트렌드 확인: ✅ 추적 가능 / ❌ 증거 부족, 당분간 추적 안 함
```

## 2단계: 공급망 물리적 분해

### 2.1 계층 분해 프레임워크

**개념 수준에 머물지 말고 물리적 실체까지 분해한다.**

```
Layer 0 (최종단): 최종 제품/서비스
    │
Layer 1 (핵심 부품): 시장이 이미 충분히 주목한 핵심 하드웨어
    │                 ⬆ 가격 반영 충분, alpha 제한적
    │─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─ ─
    │                 ⬇ 주목도 낮음, alpha 집중 구역
    │
Layer 2 (하위 부품/소재): 핵심 부품을 지탱하는 부품과 소재
    │
Layer 3 (상류 장비/원료): 하위 부품 제조에 필요한 장비와 원자재
    │
Layer 4 (인프라): 전력, 냉각, 토지, 인재, 인증
```

### 2.2 AI 인프라를 예로 든 분해 템플릿

```
Layer 0: AI 모델 학습/추론 서비스
Layer 1: GPU/가속기, HBM 메모리, 서버, 데이터센터
Layer 2 (중점 스캔 구역):
  ├─ 네트워크 인터커넥트: 광모듈, 광섬유, 스위치 칩, 구리 케이블
  ├─ 광통신 핵심: 레이저(EML/VCSEL/CW), 변조기, 광검출기
  ├─ 반도체 소재: InP 기판, GaAs 기판, SOI 웨이퍼, SiC 기판
  ├─ 첨단 패키징: CoWoS 기판, HBM TSV, ABF 필름
  ├─ PCB/기판: 고주파 고속 PCB, IC 기판, 특수 유리섬유 원단
  ├─ 테스트: 웨이퍼 레벨 테스트(Probe Card), 번인 테스트, ATE
  ├─ 방열/냉각: 액체냉각 시스템, CDU, 침수냉각액
  └─ 전력 연결: 버스덕트, UPS, 배전반, 변압기
Layer 3:
  ├─ 에피택시 장비: MOCVD, MBE
  ├─ 노광/식각: 특수 파장 노광, InP 식각
  ├─ 원자재: 고순도 금속(인듐, 갈륨, 게르마늄), 특수가스, 타겟재
  └─ 인증/표준: MSA 표준, Telcordia 인증
Layer 4:
  ├─ 전력: 원전, 천연가스 발전, 송변전
  ├─ 냉각수/방열 인프라
  └─ 데이터센터 토지/인허가
```

## 3단계: 병목 식별 — "목구멍" 찾기

### 3.1 병목 판정 6가지 기준

Layer 2-3의 각 고리에 대해 하나씩 평가한다:

| # | 기준 | 질문 | 점수 |
|---|------|------|------|
| 1 | **공급 집중도** | 글로벌 공급사 ≤3개? | 🔴 ≤2개 / 🟡 3-5개 / 🟢 >5개 |
| 2 | **증설 주기** | 신규 생산능력에 얼마나 걸리나? | 🔴 >2년 / 🟡 1-2년 / 🟢 <1년 |
| 3 | **대체 난이도** | 다른 기술/소재로 대체 가능한가? | 🔴 대체 불가 / 🟡 부분 대체 가능 / 🟢 대체 용이 |
| 4 | **가동률** | 현재 생산능력 가동률은? | 🔴 >90% / 🟡 70-90% / 🟢 <70% |
| 5 | **수요 증가율** | 하류 수요 증가율은? | 🔴 >50%/년 / 🟡 20-50% / 🟢 <20% |
| 6 | **고객 인증 주기** | 신규 공급사 진입 인증에 얼마나 걸리나? | 🔴 >1년 / 🟡 6-12개월 / 🟢 <6개월 |

**병목 등급**:
- 🔴 ≥4개 → **S급 병목** (단일 장애점 수준, 최고 우선순위)
- 🔴 3개 → **A급 병목** (심각한 제약)
- 🔴 1-2개 → **B급 병목** (압박은 있으나 통제 가능)
- 🔴 없음 → 병목 아님, 건너뜀

### 3.2 병목 지도 출력

```
공급망 병목 지도 — {트렌드 이름}
갱신 날짜: YYYY-MM-DD

S급 병목 (단일 장애점):
  1. [고리 이름] — [이유 한 문장] — 공급사: [회사 목록]
  2.

A급 병목 (심각한 제약):
  1.
  2.

B급 병목 (압박 존재):
  1.
  2.

최근 변화 (지난 스캔 대비):
  - [신규/승급/강등/해제] [고리 이름] — [이유]
```

## 4단계: 기업 선별 — 병목에서 종목으로

### 4.1 각 S급·A급 병목에 대해 관련 상장사를 전부 찾는다

검색 방식:
- WebSearch "{병목 고리} supplier listed company"
- WebSearch "{병목 고리} manufacturer stock"
- WebSearch "{병목 제품} market share company"

### 4.2 1차 선별 기준 (빠른 필터)

| 기준 | 요구사항 | 이유 |
|------|------|------|
| 상장 상태 | 상장됨 (한국/중국/홍콩/미국/일본/대만/유럽) | 거래 가능 |
| 병목 사업 비중 | 매출의 >30%가 병목 고리에서 발생 | 순수도 |
| 시가총액 | <100억 달러 우선 | 대형주는 이미 충분히 가격 반영 |
| 유동성 | 일평균 거래대금 >100만 달러 | 진입·이탈 가능 |

### 4.2.1 밸류에이션 점검 (필수 실행, 생략 불가)

**병목이 진짜라는 것 ≠ 투자 기회.** 모든 기업의 PSR, PER을 계산해 보고서에 표기해야 한다. 아래 복합 조건으로 밸류에이션의 선반영 여부를 판단한다:

#### 밸류에이션 빨간불 (하나라도 충족 → 신호 강도 상한 ★★, "⚠️ 밸류에이션 선반영" 표기)

1. **시가총액 > TAM의 20%**: 회사 시총이 도달 가능 시장 규모의 20%를 넘었다면 성장 기대가 이미 과도하게 내재화됨
2. **PSR > 30x 이고 매출 증가율 < 100%**: 고평가인데 증가율이 뒷받침을 못 함. 증가율 >100%인 회사는 PSR 레드라인을 면제하되 "⚠️ 고평가, 고성장의 지속 검증 필요" 표기
3. **시가총액 > 5년 낙관적 매출 전망의 10배**: 가장 낙관적인 가정이 전부 실현되어도 현재 가격이 여전히 과도
4. **유상증자 후 60일 내 주가 2배**: 명백한 심리 주도 특징, 신호 강도 한 등급 하향

#### 밸류에이션 노란불 (추가 설명이 필요, 없으면 강등)

1. **적자 + PSR > 15x**: ★★★ 진입은 허용하되 흑자 전환 경로와 시간표를 반드시 설명
2. **PSR이 흑자 동종업계의 5배 이상**: 프리미엄의 원천(시장 점유율, 성장률 격차, 장벽 차이)을 반드시 설명
3. **PER > 80x**: PEG를 계산하고 성장률이 뒷받침하는지 설명 필요

#### 밸류에이션 초록불 (가점 요인)

- PSR < 10x 이고 매출 성장 중 → 신호 강도 한 등급 상향 가능
- PER < 30x 이고 해자 보유 → "밸류에이션에 안전마진 있음" 표기

#### 밸류에이션 합리성 검증 (필수)

각 종목에 대해 답한다: "현재 시총에 매수하고, 가장 낙관적인 시나리오가 전부 실현되며, 10년 후 25x PER에 매도한다면 연환산 수익률은?" 연수익률 <10% → "현재 가격에 안전마진 없음" 표기.

**주의**: 밸류에이션 점검의 목적은 "PSR 100배짜리 적자 기업 추천" 같은 명백한 오류를 막는 것이지, 모든 고평가 초기 기업을 배제하는 게 아니다. 핵심은 성장률, TAM, 경쟁 구도가 현재 밸류에이션을 지탱하는지이며, 일률적 재단이 아니라 구체적 분석이 필요하다.

### 4.3 심층 선별 차원

1차 선별을 통과한 기업을 하나씩 평가한다:

```
## {회사명} ({코드})

**병목 포지셔닝**:
- 공급망에서의 구체적 위치
- 시장 점유율: 글로벌 X위, XX%
- 고객 명단 (알려진 것)

**생산능력과 확장**:
- 현재 생산능력 / 가동률
- 증설 계획 / 시간표
- 증설 필요 자금 vs 보유 현금

**재무 스냅샷**:
- 시가총액 / 매출 / 이익 / 성장률
- 병목 사업 매출 비중
- 매출총이익률 추세 (병목이 빡빡할수록 이익률은 올라가야 함)

**리스크 체크리스트**:
- [ ] 대체 기술 리스크: 우회 가능한가?
- [ ] 희석 리스크: 대규모 증자/전환사채가 있는가?
- [ ] 지정학 리스크: 민감 지역 소재/수출 통제 대상인가?
- [ ] 경영진 리스크: 불량 기록이 있는가?
- [ ] 고객 집중 리스크: 단일 고객에 과도하게 의존하는가?
- [ ] 밸류에이션 선반영: 현재 밸류에이션이 3년치 성장을 이미 반영했는가?

**병목 지속성 판단**:
- 이 병목은 언제 해소되는가?
- 해소된 후 이 회사에는 무엇이 남는가?
- 일회성인가 지속적인가?
```

## 5단계: 교차 검증 — 하나의 이야기만 듣지 않는다

### 5.1 순방향 검증

| 검증 항목 | 질문 | 검색 방식 |
|--------|------|---------|
| 고객 검증 | 선두 고객이 이미 계약/도입했는가? | 회사 공시, 고객 실적 언급 검색 |
| 매출 검증 | 병목이 이미 매출 성장에 나타났는가? | 최근 2-3개 분기 실적 검색 |
| 가격 검증 | 제품 가격이 오르고 있는가? | 산업 시세, 애널리스트 리포트 검색 |
| 생산능력 검증 | 생산능력이 정말 빡빡한가? | 납기 데이터, 고객 불만 검색 |
| 자본 검증 | 증설 자본지출이 있는가? | 회사 capex 가이던스 검색 |

### 5.2 역방향 검증 (멍거식 부정)

| 역방향 질문 | 의미 |
|---------|------|
| 똑똑한 사람들은 왜 이 주식을 안 사는가? | 알려진 약세 논점 찾기 |
| 이 병목은 우회될 수 있는가? 어떤 대체 경로가 있는가? | 기술 노선 리스크 |
| 중국/다른 플레이어가 생산능력을 빨리 복제할 수 있는가? | 공급 충격 리스크 |
| 최종 수요가 50% 둔화되면 이 회사는 어떻게 되는가? | 하방 민감도 |
| 경영진이 과거 고점에서 증자해 희석한 적이 있는가? | 경영진 신뢰도 |
| 현재 밸류에이션에 어떤 성장 가정이 내포되어 있는가? | 밸류에이션 합리성 |

### 5.3 신호 교차 검증

- 같은 병목의 여러 회사가 다 오르고 있는가? (산업 검증)
- 하류 고객이 실적 발표에서 공급 부족을 언급하는가? (고객 검증)
- 산업 협회/연구 기관에 관련 데이터가 있는가? (제3자 검증)

## 6단계: 출력 — 병목 기회 대시보드

### 6.1 병목 기회 순위표

| 순위 | 회사 | 코드 | 시총 | 연매출 | PSR | PER | 병목 고리 | 병목 등급 | 시장 점유율 | 매출 증가율 | 신호 강도 | 밸류에이션 판단 |
|------|------|------|------|--------|-----|-----|---------|---------|---------|---------|---------|---------|
| 1 | | | | | x | x | | S/A | | | ★1-5 | 적정/다소 높음/선반영 |

**필수 필드**: 시총, 연매출, PSR, PER은 필수이며 "확인 예정"으로 건너뛸 수 없다. 재무 데이터를 구할 수 없으면 신호 강도는 ★★를 넘을 수 없다.

신호 강도 등급 (밸류에이션 점검 결과가 등급에 직접 영향):
- ★★★★★ 다중 교차 검증, 고객 도입 완료, 매출에 이미 반영, 밸류에이션 초록불 (PSR 적정 + 흑자 또는 흑자 근접)
- ★★★★ 대부분 검증 통과, 밸류에이션 초록불 또는 노란불 (설명 첨부 필요)
- ★★★ 논리는 성립하나 일부 검증 대기, 밸류에이션 노란불 수용 가능 (고성장 초기 기업 등)
- ★★ 초기 신호, 또는 병목 논리는 성립하나 밸류에이션 빨간불 (시총>TAM 20%, PSR>30x에 성장 부족, 시총이 5년 전망을 크게 초과 등)
- ★ 순수 개념, 미검증

### 7.1 실행할 때마다의 증분 업데이트

1. 이미 식별된 병목이 여전히 유효한지 확인

   - 대체 기술에 돌파가 있었는가?

2. 새로 나타난 병목 스캔
   - 최근 7일의 supply chain / shortage / bottleneck 뉴스 검색
   - 실적 시즌의 공급망 관련 공시 확인

3. 병목 등급 갱신 (승급/강등/해제)

## 매시간 스캔 모드 (정기 작업용)

매시간 1회 실행하며 "건질 게 있을 때만 보고서를 내는" 모드를 채택한다:

### 스캔 절차 (매시간)

1. **뉴스 스캔**: 지난 1-2시간의 공급망 관련 뉴스 검색
   - 키워드: supply chain bottleneck, shortage, capacity constraint, allocation, lead time, sole source, 병목, 품절, 생산능력, 가격 인상
   - 커버리지: 영어 + 아시아 언어 출처
2. **시장 신호**: 추적 중인 기업의 주가 변화 확인 (특히 >5% 이상 변동에 주목)
3. **실적/공시**: 병목 관련 기업의 실적 발표나 중대 공시 확인
4. **밸류에이션 기회**: watchlist 기업이 시장 하락 등으로 매수 구간에 진입했는지 확인
5. **보고서 발행 여부 판단**:
   - 새 병목 신호, 명확한 종목 기회, 중대 상태 변화 → **보고서 발행**
   - 새 발견 없음 → **보고서 미발행**, 로그에 "이번 라운드 새 신호 없음"만 기록

## AI 리서치 편향 자각

| 편향 | 양상 | 대응 |
|------|-----|------|
| 선두 기업 선호 | 검색 결과가 대형주로 채워짐 | 소형 공급사를 의도적으로 검색, "small cap" 키워드 추가 |
| 영어 선호 | 일본·한국·대만 기업 누락 | 반드시 일본/한국/대만 시장의 공급사를 검색 |
| 내러티브 선호 | "AI 테마" 라벨에 끌림 | 시장 라벨이 아니라 실제 공급망 위치만 본다 |
| 확증 편향 | 병목을 찾은 후 긍정적 증거만 찾음 | 역방향 검증 강제 실행 (5단계) |
| 시효 편향 | 낡은 정보에 의존 | 최근 30일 데이터를 우선 검색 |

## 핵심 원칙 (최고 우선순위)

1. **AI에게 주식 추천을 시키지 말고 공급망을 분해시켜라** — 질문이 답보다 중요하다
2. **물리 우선** — 실제 물리적 제품/소재/장비가 필요한 고리에만 주목
3. **2층, 3층** — 이미 충분히 가격 반영된 선두 기업을 쫓지 않는다
4. **교차 검증** — 모든 결론에 최소 2개의 독립 출처
5. **불확실성을 정직하게 마주한다** — 데이터를 못 찾으면 "데이터 부족"이라 쓰고 추측으로 채우지 않는다
6. **병목에는 시효가 있다** — 모든 병목은 언젠가 해소된다. 핵심은 시간 창의 판단이다
7. **소형주 ≠ 좋은 기회** — 소형주도 나쁜 회사일 수 있다. 반드시 재무 품질 관문을 통과시켜라
8. **병목이 진짜 ≠ 투자 기회** — 가장 빡빡한 병목 위에 앉은 회사라도 PSR>30x이거나 적자라면 현재 가격은 매수점이 아니다. **밸류에이션은 하드 게이트다. 병목 순수도, 신호 강도, 내러티브 매력으로 덮을 수 없다.** 오른 병목주를 놓칠지언정 100배 PSR에 적자 기업을 사지 않는다
9. **CLAUDE.md 객관성 원칙 준수** — 강세를 미리 정하지 않는다. 데이터 먼저, 결론은 나중
