# 포트폴리오 관리: "기업 리서치"에서 "포트폴리오 운용"으로

$ARGUMENTS 에 대해 포트폴리오 점검과 최적화를 수행한다.

**지원하는 입력 형식**:
- 보유 종목 리스트. 예: `텐센트 30%, 메이투안 20%, 마오타이 20%, 엔비디아 15%, 현금 15%`
- 또는: `텐센트 500주 @480홍콩달러, 메이투안 1000주 @130홍콩달러, ...`
- 또는: `내 보유 종목` (저장된 포트폴리오 파일 `reports/portfolio-latest.md`가 이미 있는 경우)

> "분산투자는 무지에 대한 보호막이다. 자신이 무엇을 하는지 안다면 분산투자는 의미가 없다." — 버핏
>
> "내 평생 본 진짜 좋은 투자 기회는 열 손가락으로 셀 수 있다." — 리루

## 설계 이념

기업 리서치는 투자의 절반일 뿐이다. 나머지 절반은 **포트폴리오 차원의 의사결정**이다:
- 얼마나 살 것인가? (비중)
- 어떤 돈으로 살 것인가? (자금 출처 — 신규 자금인가 교체매매인가)
- 기존 보유 종목과 충돌하지 않는가? (상관관계)
- 최적의 포트폴리오는 어떤 모습인가? (기회비용)

버핏은 결코 한 종목을 고립적으로 보지 않는다 — 그는 항상 "이것이 내가 할 수 있는 최선인가?"를 생각한다.

## 실행 절차

### 1단계: 보유 종목 해석

입력에서 현재 보유 종목을 해석해 아래 형식으로 표준화한다:

| 종목 | 코드 | 보유량 | 매입가 | 현재가 | 평가액 | 비중 | 손익 |
|------|------|--------|-------|------|------|------|------|

입력에 비율만 있고 금액이 없으면 비율 기준으로 분석하면 된다.

동시에 기존 포트폴리오 파일(`reports/portfolio-latest.md`)이 있는지 확인하고, 있으면 읽어서 업데이트한다.

### 2단계: 최신 데이터 수집

Task 도구로 백그라운드 Agent를 띄워, WebSearch로 각 보유 종목에 대해 병렬 수집한다:
1. 현재 주가와 밸류에이션 지표 (PER, PBR, 배당수익률)
2. 최근 분기의 핵심 재무 변화
3. 최근 중대 이벤트
4. 애널리스트 컨센서스 (선행 PER, 목표 주가)

각 보유 종목에 대해 `tools/financial_rigor.py verify-valuation`으로 밸류에이션 데이터를 검증한다. 종목별로 정보 풍부도(A/B/C등급)를 표기하고, C등급 종목의 분석 결론에는 낮은 신뢰도를 명시한다.

### 3단계: 개별 포지션 건강검진

각 보유 종목에 대해 빠른 건강 점검을 수행한다:

| 종목 | 현재 PER | 매수 논리 변화 여부 | 투자논지 건강도 | 비중 제안 |
|------|:------:|:--------------:|:---------:|---------|
| 텐센트 | 18x | 변화 없음 | 8/10 | 적정 |
| 메이투안 | 25x | 경쟁 심화 | 6/10 | 다소 높음, 비중 축소 검토 |

각 보유 종목에 대해 답한다:
- [ ] **오늘 보유하고 있지 않다면, 현재 가격에 다시 사겠는가?**
- [ ] **내일부터 거래가 불가능하다면, 5년 보유해도 마음이 편한가?**
- [ ] **매수 논지는 아직 온전한가?**

**돤융핑**: "한 주식을 10년 보유할 생각이 없다면 하루도 보유하지 마라."

### 4단계: 포트폴리오 차원 분석

#### 4.1 집중도 분석

| 지표 | 현재값 | 권장 범위 | 판단 |
|------|-------|---------|------|
| 최대 보유 종목 비중 | | <40% | |
| 상위 3종목 비중 | | 50-80% | |
| 총 보유 종목 수 | | 5-15개 | |
| 현금 비중 | | 10-30% (시장 환경에 따라) | |

**리루의 기준**: 핵심 보유 3-5종목, 상위 3종목이 80%+. **단, 이는 종목 하나하나를 철저히 연구했다는 전제가 필요하다.**

**버핏의 기준**: 핵심 보유는 10종목 이하, 다만 더 많은 위성 포지션은 허용.

#### 4.2 상관관계 점검

보유 종목 간의 숨은 연관성을 식별한다:

| 보유 A | 보유 B | 연관 유형 | 리스크 |
|-------|-------|---------|------|
| 텐센트 | 콰이쇼우 | 둘 다 중국 인터넷 | 규제 리스크 공진 |
| 엔비디아 | TSMC | AI 공급망 상하류 | AI Capex 동반 변동 |
| 메이투안 | 핀둬둬 | 둘 다 중국 소비 | 거시 소비 동반 변동 |

**체크리스트**:
- [ ] 전체 비중의 50% 이상이 같은 테마/산업에 노출되어 있는가?
- [ ] 전체 비중의 50% 이상이 같은 국가/통화에 노출되어 있는가?
- [ ] 미중 관계가 악화되면 포트폴리오는 얼마나 손실을 보는가?
- [ ] 글로벌 경기 침체가 오면 포트폴리오는 얼마나 손실을 보는가?

#### 4.3 기회비용 분석

이것이 버핏의 가장 핵심적인 사고방식이다 — **모든 1원은 수익률이 가장 높은 곳에 있어야 한다**.

모든 보유 종목을 "기대 연환산 수익률" 순으로 정렬한다:

| 순위 | 종목 | 현재 비중 | 기대 연수익률 | 확실성 | 기대수익×확실성 |
|:----:|------|:-------:|:----------:|:------:|:--------------:|
| 1 | | | | | |
| 2 | | | | | |
| ... | | | | | |

기대수익률 추정 방법 (`tools/financial_rigor.py three-scenario`로 계산):
- **간이 공식**: 기대 연수익 ≈ FCF Yield + 기대 성장률 (주 방법)
- **가치형 검증**: 안전마진 회귀 + 이익 성장률 + 배당수익률
- **성장형 검증**: 이익 성장률 × 적정 PER 변화

**핵심 질문**: 최하위 보유 종목의 기대수익률이 현금(무위험 수익률 ~4%)보다 높은가? 아니라면 팔아서 현금으로 바꿔야 한다.

#### 4.4 스트레스 테스트

| 시나리오 | 가정 | 포트폴리오 예상 영향 | 최대 낙폭 |
|------|------|-----------|---------|
| 글로벌 침체 | 기업 이익 20-30% 감소 | | |
| 미중 갈등 격화 | 중국 관련주 50% 디스카운트 | | |
| 금리 급등 | 10년물 국채 → 6% | | |
| 기술주 버블 붕괴 | 기술주 PER 40% 압축 | | |

각 시나리오에 대해 정성 + 대략적 정량 평가를 한다 (각 종목의 산업 특성과 과거 밸류에이션 변동 범위 기준):
- 어떤 종목이 가장 큰 충격을 받는가? 대략적인 영향 방향과 규모 범위
- 포트폴리오 전체가 견딜 수 있는가?
- 헤지가 필요한가?

### 5단계: 최적화 제안

#### 5.1 리밸런싱 제안

위 분석을 바탕으로 구체적인 리밸런싱 제안을 준다:

| 액션 | 종목 | 현재 비중 | 제안 비중 | 이유 |
|------|------|:-------:|:-------:|------|
| 비중 확대 | | | | |
| 비중 축소 | | | | |
| 전량 매도 | | | | |
| 신규 편입 | | | | |
| 유지 | | | | |

#### 5.2 대체 종목 탐색

포트폴리오에 "현금보다 못한" 포지션이 있거나 현금 비중이 지나치게 높으면, 이 Skill 안에서 개별 종목을 직접 추천하지 말고 `/industry-research` 또는 `/investment-checklist`로 관심 산업/기업을 체계적으로 스크리닝하도록 권한다.

#### 5.3 현금 관리

| 현재 현금 비중 | 권장 현금 비중 | 이유 |
|:----------:|:----------:|------|

**버핏**: 현재 3,820억 달러의 현금을 보유하고 있고 총자산의 25%를 넘는다 — 좋은 기회를 찾지 못했을 때는 현금이 최고의 포지션이다.

### 6단계: 포트폴리오 보고서 출력

#### 보고서 구조

```
1. 포트폴리오 개요 (보유 표 + 파이차트 설명)
2. 개별 포지션 건강검진 (각 종목의 건강 상태)
3. 포트폴리오 분석
   - 집중도: 과도한 분산/집중은 아닌가?
   - 상관관계: 숨은 연관성과 리스크 공진
   - 기회비용: 최하위 포지션은 보유할 가치가 있는가?
   - 스트레스 테스트: 극단 시나리오에서의 낙폭 추정
4. 리밸런싱 제안 (구체적 조치 + 이유)
5. 다음 점검 시기와 관찰 포인트
```

#### 결론은 반드시 명확히 답해야 한다

1. **포트폴리오 전체 건강도**: 우수 / 양호 / 조정 필요 / 문제 심각
2. **지금 가장 해야 할 한 가지는 무엇인가?** (X 비중 확대 / Y 비중 축소 / 유지)
3. **현재 가장 큰 리스크는 무엇인가?**

### 7단계: 포트폴리오 파일 저장

포트폴리오 정보를 `reports/portfolio-latest.md`에 기록한다. 포함 내용:
- 최신 보유 종목 표
- 이번 점검 날짜와 결론
- 리밸런싱 기록 (누적 추가)
- 다음 점검 알림

## 핵심 원칙

- **모든 1원에는 기회비용이 있다** — 평범한 주식을 보유하는 비용은 훌륭한 주식을 놓치는 것이다
- **집중이 위험이 아니라 무지가 위험이다** — 깊이 이해하는 3종목 보유가 어설프게 아는 30종목 보유보다 안전하다
- **현금도 하나의 포지션이다** — 좋은 기회가 없을 때 현금을 들고 있는 것은 부끄러운 일이 아니다
- **포트폴리오 차원 > 개별 종목 차원** — 좋은 주식도 잘못된 비중에서는 발목을 잡는다
- **정기 점검하되 과도하게 거래하지 마라** — 분기 1회 점검이면 충분하다. 매일 시세를 보며 매매하지 마라
