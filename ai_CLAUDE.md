## 사용자 프로필

- 투자 스타일: 가치투자, 집중 보유
- 리서치 선호: 직접적이고 예리하게, 군말 없이. 양다리 걸치지 말고 명확한 결론. 데이터는 반드시 정확하게
- 사용 시나리오: 개인 투자 의사결정 보조
- 언어: 한국어 (원본 프로젝트의 중국어 콘텐츠를 한국어로 전환함)

## 프로젝트 진화 역사 (업스트림에서 계승)

### 2026년 4월 7-9일 (첫 리서치 + 프레임워크 정비)

**완료된 리서치:**
1. `/investment-team 핀둬둬` — 첫 완전한 4 Agent 병렬 리서치, 종합 점수 3.4/5
2. `/investment-checklist` 7개 기업 — 마오타이, 텐센트, 엔비디아, 메이투안, 핀둬둬, 팝마트, 콰이쇼우
3. 거장 보유 추적 — 버핏/리루/돤융핑 최신 13F + PDD 매입 단가 분석
4. 메이투안 등 5개 기업 심층 재평가 (사용자가 초기 평가에 이의 제기)

**사용자 피드백으로 인한 수정:**
- 메이투안: ❌ → ✅ 조건부 통과 — 사용자 지적: 흑자 회복을 기다렸다 사면 늦다. 2,000억을 쏟아부어도 못 무너뜨린 것 = 진짜 해자
- 엔비디아: ❓ → ✅ 조건부 통과 — AI Capex는 여전히 가속 중, Jevons Paradox
- 콰이쇼우: ❓ → ✅ 조건부 통과 — Kling AI 저평가, Sora는 중단됨

**핵심 교훈:**
- checklist를 기계적으로 적용하지 말고 독립적으로 판단할 것
- "흑자 회복을 기다렸다 산다"는 논리적 오류다 — 주가는 선반영된다
- 경쟁사가 더 많은 돈을 쓰고도 이득을 못 봤다 = 해자의 최고 증거

## 프로젝트 핵심 강점 (README에 반영됨)

1. **결론을 강제하고 얼버무리지 않는다** — 통과/미통과/회색, 구체적 가격 구간 포함
2. **4대 거장 관점의 대립** — 분업이 아니라 상호 도전, 진짜 모순과 긴장을 만든다
3. **구조화된 반편향 메커니즘** — A/B/C 정보 풍부도, 멍거 역발상, 빠른 부결, 반컨센서스
4. **금융 데이터 정밀성** — Decimal 정밀 계산, 시총 수동 검산, 다중 출처 교차 검증
5. **재현 가능한 리서치 프로세스** — 같은 입력 → 일관된 구조의 출력, 횡단 비교와 종단 추적 지원
6. **멀티 Agent 병렬 심층** — 4개 Agent가 각자 검색 + 독립 분석, 4배의 정보량
7. **실전 검증** — (업스트림 원저자의 기록) 2년 누적 146만 위안 수익, 지수를 40-50%p 연속 아웃퍼폼

## 사용자 선호와 작업 습관

- **보고서 언어**: 한국어
- **GitHub 푸시**: 리서치 완료 후 푸시 여부를 능동적으로 질문
- **git 작업**: 원격에 새 커밋이 있을 수 있으므로 푸시 전 반드시 `git pull --rebase`
- **오류에 대한 태도**: 직접 지적하고 돌려 말하지 않는다. 사용자가 AI의 판단에 이의를 제기하면 방어하지 말고 진지하게 재평가한다
- **과잉 요약 금지**: 사용자는 diff를 볼 수 있으니 작업할 때마다 한 일을 되풀이해 설명할 필요 없다
- **리서치 깊이**: 빠르게 대충 하느니 시간을 들여 깊고 정확하게 한다
